Suivi des travaux - version locale

Cette version est adaptée pour fonctionner sur un PC classique avec React + Vite.
La sauvegarde se fait dans le navigateur via localStorage.

Installation :
1. Installer Node.js
2. Ouvrir un terminal dans ce dossier
3. Exécuter : npm install
4. Puis : npm run dev
5. Ouvrir l'adresse affichée dans le navigateur

Notes :
- Les données restent stockées localement dans le navigateur utilisé.
- Le fichier original dépendait de window.storage, qui semble être un environnement spécial et non standard.
